===============================================================
Using dictionary learning to obtain clean resting-state network
===============================================================

.. topic:: **Page summary**

   This page demonstrate the use of dictionary learning methods over resting
   state fMRI data, and compare it to standard Independant Component Analysis,
   described (:ref:`here <extracting_rsn>`). To understand this section, ICA
   examples should have already been mastered.

.. topic:: **References**

   * G. Varoquaux et al.
    "Multi-subject dictionary learning to segment an atlas of brain spontaneous
    activity",
    `Information Processing in Medical Imaging (2011)
    <https://hal.inria.fr/inria-00588898v2/document>`_, p. 562-573
